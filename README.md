# 📈 PaperTrade — Stock Market Simulator

A full-featured desktop stock trading simulator built with **Java Swing**. Practice trading with $10,000 virtual cash, track your portfolio in real time, and experience a realistic market environment — all without any financial risk.

![License](https://img.shields.io/badge/License-MIT-green)

---

## ✨ Features

- 💰 **Virtual Trading** – Start with $10,000 paper money and trade 10+ popular stocks (AAPL, MSFT, GOOGL, TSLA, etc.)
- 📊 **Live Market Simulation** – Prices update every 3 seconds with realistic random walks.
- 📈 **Interactive Charts** – View price trends over 1D, 1W, 1M, 3M, and 1Y timeframes.
- 💼 **Portfolio Dashboard** – Track total value, P&L, daily gain/loss, and best performer.
- 🛒 **Buy / Sell Orders** – Place orders with quantity spinners and instant feedback.
- 💳 **Payment Gateway** – Add funds via simulated Credit/Debit Card or UPI (with QR code).
- 📜 **Order History** – Complete transaction log with timestamps and status.
- 🎨 **Dark Themed UI** – Modern, sleek dark interface optimized for trading.
